HTML - это язык разметки, используемый для создания веб-страниц. Он определяет структуру содержимого страницы, такую как заголовки, параграфы и изображения, с помощью тегов. Браузеры интерпретируют HTML и отображают веб-страницы в соответствии с ее структурой.

Ссылка на HTML

CSS - это язык стилей, который определяет внешний вид и оформление элементов на веб-странице. С помощью CSS можно задавать цвета, шрифты, отступы, размеры и расположение элементов, чтобы придать странице желаемый внешний вид и стиль. Браузеры интерпретируют CSS и применяют стили к элементам в соответствии с заданными правилами.

Ссылка на CSS

JavaScript - это язык программирования, который добавляет интерактивность на веб-страницы. Он позволяет создавать динамические элементы, обрабатывать события пользователя и взаимодействовать с содержимым страницы. Браузеры выполняют JavaScript для создания динамических и интерактивных функций на веб-страницах.
